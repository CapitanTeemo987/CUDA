# Programación Paralela con CUDA

Este repositorio contiene una serie de programas desarrollados en **CUDA C/C++** para aplicar conceptos de programación paralela sobre GPU utilizando la plataforma NVIDIA CUDA.

## Contenido

- **Suma de números primos**
  - Calcula la suma de todos los números primos menores a un límite utilizando reducción paralela y operaciones atómicas.

- **Aproximación de π**
  - Implementa el cálculo de π mediante la serie de Leibniz aprovechando la ejecución paralela en GPU.

- **Contador de números pares**
  - Cuenta la cantidad de números pares en un arreglo utilizando kernels de CUDA.

## Tecnologías utilizadas

- CUDA
- C++
- NVIDIA CUDA Runtime API

## Conceptos aplicados

- Ejecución paralela de kernels
- Memoria compartida (Shared Memory)
- Reducción paralela
- Operaciones atómicas
- Sincronización de hilos
- Gestión de memoria entre CPU y GPU
